# DOMPP Step-by-Step Case Study (JSX)

The JSX version of the step-by-step case study.

It uses a tiny custom JSX factory (see `main.jsx`) and a bootstrap file that installs DOMPP, reconcile, and stateful.

---

## Files

* `index.html` demo shell
* `bootstrap.js` installs DOMPP and addons
* `main.jsx` JSX version of the demo

---

## How to Run

From the project root:

```bash
node examples/serve.js
```

Then open `http://localhost:3000` and select this example.
