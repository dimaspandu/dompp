# Core Setter Callback

This example shows callback updaters in core DOM++ without the stateful addon.

Setters demonstrated:

- `setText(({ text }) => ...)`
- `setStyles(({ styles }) => ...)`
- `setAttributes(({ attributes }) => ...)`
- `setEvents(({ events }) => ...)`
- `setChildren(({ childNodes }) => ...)`

## Run

```bash
node examples/serve.js
```

Open:

`http://localhost:3000/examples/setter-callback-core/`
