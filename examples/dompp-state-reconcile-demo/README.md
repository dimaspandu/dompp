# DOMPP State + Reconcile Demo

This example focuses on how `stateful` and `reconcile` interact.

It shows:

* Local element state with `setState`
* `matchById` preserving DOM identity
* Cross-element updates (explicit state mutation)
* Node reuse by position when ids are missing

---

## Files

* `index.html` demo shell
* `main.js` example logic

---

## How to Run

From the project root:

```bash
node examples/serve.js
```

Then open `http://localhost:3000` and select this example.
