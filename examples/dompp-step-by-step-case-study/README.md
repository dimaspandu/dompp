# DOMPP Step-by-Step Case Study

A guided example that builds up DOMPP features in small steps.

It demonstrates:

* Basic DOM++ chaining
* Element-local state with `setState`
* Cross-element updates
* `matchById` reconciliation preserving DOM identity

---

## Files

* `index.html` demo shell
* `main.js` example logic

---

## How to Run

From the project root:

```bash
node examples/serve.js
```

Then open `http://localhost:3000` and select this example.
